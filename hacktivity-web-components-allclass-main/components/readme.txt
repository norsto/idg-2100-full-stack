Use this folder to store the created components

This folder contains all the components of your project. 

- `coundown-component.js` contains the code of the component used to start the game
- `game-with-player-component.js` contains the code of that shows the player name and listens to players to press a key.
- `winner-component.js` contains the component that waits for the game to finish and announce the winner.
